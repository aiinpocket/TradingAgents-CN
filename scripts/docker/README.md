# Docker Scripts

## 目錄說明

Docker容器管理腳本

## 腳本列表

- `docker-compose-start.bat - 啟動Docker Compose`
- `start_docker_services.* - 啟動Docker服務`
- `stop_docker_services.* - 停止Docker服務`
- `mongo-init.js - MongoDB初始化腳本`

## 使用方法

```bash
# 進入項目根目錄
cd C:\code\TradingAgentsCN

# 運行腳本
python scripts/docker/script_name.py
```

## 註意事項

- 確保在項目根目錄下運行腳本
- 檢查腳本的依賴要求
- 某些腳本可能需要特殊權限
