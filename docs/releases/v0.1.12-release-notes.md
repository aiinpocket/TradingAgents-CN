# TradingAgents-CN v0.1.12 發布說明

## 🚀 版本概述

**發布日期**: 2025年7月29日  
**版本號**: cn-0.1.12  
**主題**: 智能新聞分析模塊全面升級與項目結構優化

v0.1.12是一個重大功能更新版本，專註於新聞分析能力的全面提升。本版本新增了完整的智能新聞分析模塊，包括多層次新聞過濾、品質評估、相關性分析等核心功能，同時對項目結構進行了全面優化，提升了代碼組織和維護性。

## 🆕 主要新功能

### 🧠 智能新聞分析模塊

#### 1. 智能新聞過濾器 (`news_filter.py`)
- **AI驅動的相關性評分**: 基於機器學習算法評估新聞與股票的相關性
- **智能品質評估**: 自動識別和過濾低品質、重複、無關新聞
- **多維度評分機制**: 從內容品質、時效性、來源可信度等多個維度評估新聞
- **靈活配置選項**: 支援自定義過濾閾值和評分權重

#### 2. 增強新聞過濾器 (`enhanced_news_filter.py`)
- **深度語義分析**: 使用先進的NLP技術進行新聞內容深度理解
- **情感傾向識別**: 自動識別新聞的情感傾向（正面/負面/中性）
- **關鍵詞提取**: 智能提取新聞中的關鍵信息和實體
- **重複內容檢測**: 高效識別和去除重複或相似的新聞內容

#### 3. 新聞過濾集成模塊 (`news_filter_integration.py`)
- **多級過濾流水線**: 基礎過濾 → 增強過濾 → 集成過濾的三級處理機制
- **智能降級策略**: 當高級過濾失敗時自動降級到基礎過濾
- **性能優化**: 緩存機制和批處理優化，提升處理效率
- **統一接口**: 為上層應用提供簡潔統一的調用接口

#### 4. 統一新聞工具 (`unified_news_tool.py`)
- **多源新聞整合**: 整合Google News、FinnHub等多個新聞源
- **統一數據格式**: 標準化不同來源的新聞數據格式
- **智能去重**: 跨數據源的新聞去重和合並
- **實時更新**: 支援實時新聞獲取和增量更新

#### 5. 增強新聞檢索器 (`enhanced_news_retriever.py`)
- **智能搜索**: 基於股票代碼、公司名稱、行業關鍵詞的智能搜索
- **時間範圍過濾**: 靈活的時間範圍設置和歷史新聞檢索
- **語言支援**: 支援中英文新聞的混合檢索和處理
- **結果排序**: 基於相關性和重要性的智能排序算法

## 🔄 升級指南

### 從 v0.1.11 升級到 v0.1.12

#### 1. 代碼更新
```bash
# 拉取最新代碼
git pull origin main

# 更新依賴
pip install -r requirements.txt
```

#### 2. 新功能使用

##### 使用智能新聞過濾器
```python
from tradingagents.utils.news_filter import NewsFilter

# 創建新聞過濾器
filter = NewsFilter()

# 過濾新聞
filtered_news = filter.filter_news(news_list, stock_symbol="AAPL")
```

##### 使用統一新聞工具
```python
from tradingagents.tools.unified_news_tool import UnifiedNewsTool

# 創建新聞工具
news_tool = UnifiedNewsTool()

# 獲取新聞
news = news_tool.get_news(symbol="000001", limit=10)
```

#### 3. 配置更新
新版本新增了新聞過濾相關的配置選項，可以在配置文件中進行自定義：

```yaml
news_filter:
  relevance_threshold: 0.6
  quality_threshold: 0.7
  enable_enhanced_filter: true
  cache_enabled: true
```
