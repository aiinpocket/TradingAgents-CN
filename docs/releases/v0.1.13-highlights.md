## 🚀 快速開始

### 1. 環境準备

```bash
# 切換到預覽版分支
git checkout feature/native-openai-support

# 更新依賴
pip install -r requirements.txt
```

### 2. 配置Google AI

```bash
# 在 .env 文件中添加
echo "GOOGLE_API_KEY=your_api_key_here" >> .env
```

### 3. 驗證安裝

```bash
# 測試Google AI功能
python tests/test_gemini_simple.py

# 啟動Web界面
streamlit run web/app.py
```

### 4. 體驗新功能

- 🎯 在Web界面中選擇Google AI模型
- 🔧 配置自定義OpenAI端點
- 📊 運行股票分析測試新功能
