# TradingAgents-CN Web界面使用指南 (v0.1.4)

## 🌐 概述

TradingAgents-CN 提供了直觀易用的Web界面，讓您可以通過瀏覽器輕鬆進行股票投資分析。v0.1.4版本新增了配置管理、Token統計等功能，本指南將詳細介紹Web界面的各項功能和配置選項。

## ✨ v0.1.4 新增功能

- 🎛️ **配置管理**: API密鑰管理、模型選擇、系統配置
- 💰 **Token統計**: 實時Token使用統計和成本追蹤
- 💾 **緩存管理**: 數據緩存狀態監控和管理
- 🌍 **國際市場支援**: 美股股票分析功能

## 🚀 啟動Web界面

### 1. 啟動命令
```bash
# 啟動Web界面
python -m streamlit run web/app.py

# 或者使用簡化命令
streamlit run web/app.py
```

### 2. 訪問地址
啟動後，在瀏覽器中訪問：`http://localhost:8501`

## 📊 界面功能詳解

### 分析配置區域

#### 基本設置
- **股票代碼**: 輸入要分析的股票代碼
  - 🇺🇸 **美股**: AAPL, TSLA, NVDA, MSFT
- **分析日期**: 選擇分析的基準日期
- **研究深度**: 選擇分析的詳細程度（1-5級）

#### 🎯 研究深度詳細說明

研究深度是控制分析質量、速度和成本的核心參數。不同級別的具體配置如下：

##### 🚀 1級 - 快速分析 (2-4分鐘)
**適用場景**: 日常快速決策、市場概覽

**技術配置**:
- **辯論輪次**: 1輪 (最少)
- **AI模型**: gemini-flash (最快)
- **記憶功能**: ❌ 禁用 (加速)
- **在線工具**: ❌ 禁用 (使用緩存數據)

**特點**:
- ✅ 速度最快、成本最低
- ✅ 適合頻繁查詢
- ❌ 分析深度有限
- ❌ 可能錯過細節資訊

##### 📈 2級 - 基礎分析 (4-6分鐘)
**適用場景**: 常規投資決策、基礎研究

**技術配置**:
- **辯論輪次**: 1輪
- **AI模型**: gemini-pro (平衡)
- **記憶功能**: ✅ 啟用
- **在線工具**: ✅ 啟用 (獲取最新數據)

**特點**:
- ✅ 速度較快、包含最新數據
- ✅ 成本可控
- ❌ 辯論深度有限

##### 🎯 3級 - 標準分析 (6-10分鐘) **[默認推薦]**
**適用場景**: 重要投資決策、標準研究流程

**技術配置**:
- **辯論輪次**: 1輪 (研究員) + 2輪 (風險評估)
- **AI模型**: gemini-pro (質量優先)
- **記憶功能**: ✅ 啟用
- **在線工具**: ✅ 啟用

**特點**:
- ✅ 平衡速度和質量
- ✅ 風險評估更深入
- ✅ 適合大多數場景
- ❌ 耗時適中

##### 🔬 4級 - 深度分析 (10-15分鐘)
**適用場景**: 重大投資決策、詳細研究報告

**技術配置**:
- **辯論輪次**: 2輪 (研究員) + 2輪 (風險評估)
- **AI模型**: gemini-pro (高質量)
- **記憶功能**: ✅ 啟用
- **在線工具**: ✅ 啟用

**特點**:
- ✅ 分析深度高
- ✅ 多輪辯論確保全面性
- ✅ 適合重要決策
- ❌ 耗時較長、成本較高

##### 🏆 5級 - 全面分析 (15-25分鐘)
**適用場景**: 最重要的投資決策、完整研究報告

**技術配置**:
- **辯論輪次**: 3輪 (研究員) + 3輪 (風險評估)
- **AI模型**: gpt-4 (最高質量)
- **記憶功能**: ✅ 啟用
- **在線工具**: ✅ 啟用

**特點**:
- ✅ 最全面的分析
- ✅ 最高質量的推理
- ✅ 最可靠的結果
- ❌ 耗時最長、成本最高

#### 📋 研究深度選擇建議

| 使用場景 | 推薦級別 | 預期耗時 | 成本 | 適用情況 |
|----------|----------|----------|------|----------|
| 日常市場監控 | 1-2級 | 2-6分鐘 | 低 | 快速獲取市場概況 |
| 常規投資決策 | 2-3級 | 4-10分鐘 | 中低 | 平衡速度和質量 |
| 重要投資決策 | 3-4級 | 6-15分鐘 | 中高 | 確保分析質量 |
| 重大資金投入 | 4-5級 | 10-25分鐘 | 高 | 最全面的風險評估 |
| 研究報告撰寫 | 4-5級 | 10-25分鐘 | 高 | 需要詳細的分析內容 |

### 分析師團隊選擇

#### 可選分析師類型
- **📊 市場技術分析師**: 技術指標、價格趨勢分析
- **💰 基本面分析師**: 財務數據、公司基本面分析
- **📰 新聞分析師**: 新聞事件、市場動態分析
- **💭 社交媒體分析師**: 市場情緒、投資者態度分析

#### 分析師組合建議
- **最快組合**: 只選擇"市場技術分析師" (最快)
- **平衡組合**: "市場技術" + "基本面分析師" (推薦)
- **全面組合**: 選擇所有分析師 (最全面但最慢)

### 高級選項

#### 模型配置
- **LLM提供商**: 選擇AI模型提供商 (Google AI/OpenAI)
- **具體模型**: 選擇使用的具體AI模型

#### 分析選項
- **包含情緒分析**: 是否包含市場情緒分析
- **包含風險評估**: 是否包含詳細風險評估
- **自訂提示**: 添加特定的分析要求

## 📈 分析結果解讀

### 核心決策指標
- **投資建議**: 買入/持有/賣出
- **目標價位**: AI預測的合理價格目標
- **置信度**: 對決策的信心程度 (0-1)
- **風險評分**: 投資風險等級 (0-1)

### 詳細分析報告
- **🧠 AI分析推理**: 決策的詳細邏輯
- **📊 技術分析**: 技術指標和趨勢分析
- **💰 基本面分析**: 財務狀況和業務分析
- **📰 新聞影響**: 相關新聞事件分析
- **💭 市場情緒**: 投資者情緒和態度
- **⚖️ 風險評估**: 詳細的風險分析

## 💡 使用技巧

### 速度優化
1. **快速查看**: 使用1-2級研究深度 + 最少分析師
2. **平衡分析**: 使用2-3級研究深度 + 核心分析師
3. **深度研究**: 使用4-5級研究深度 + 全部分析師

### 成本控制
1. **日常監控**: 使用快速模式，降低API調用成本
2. **重要決策**: 使用標準模式，確保分析質量
3. **關鍵投資**: 使用深度模式，獲得最可靠結果

### 結果驗證
1. **多次分析**: 對重要股票進行多次分析對比
2. **不同深度**: 使用不同研究深度驗證一致性
3. **歷史回測**: 查看歷史分析的準確性

## 🔧 故障排除

### 常見問題
1. **分析時間過長**: 降低研究深度或減少分析師數量
2. **目標價位顯示N/A**: 檢查AI模型配置和API連接
3. **分析失敗**: 檢查API密鑰配置和網路連接

### 性能優化
1. **使用緩存**: 啟用數據緩存功能
2. **選擇合適模型**: 根據需求選擇速度和質量平衡的模型
3. **合理配置**: 根據使用場景選擇合適的研究深度
