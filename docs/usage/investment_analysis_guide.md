# TradingAgents-CN 投資分析使用指南

## 🎯 概述

TradingAgents-CN 是一個基於多智能體大語言模型的投資分析框架，能夠為您提供專業的股票分析報告。

## 🚀 快速開始

### 1. 基礎配置

確保您已經配置了必要的API密鑰：

```bash
# 檢查配置狀態
python -m cli.main config

# 運行集成測試
python -m cli.main test
```

### 2. 使用方式選擇

#### 🌐 Web界面 (推薦新手)
```bash
# 啟動Web界面
python -m streamlit run web/app.py
```
然後在瀏覽器中訪問 `http://localhost:8501`

**優點**:
- 直觀易用的圖形界面
- 實時進度顯示
- 詳細的配置選項
- 結果視覺化展示

**詳細使用說明**: 請參考 [Web界面使用指南](web-interface-guide.md)

#### 💻 命令列界面 (適合開發者)
```bash
# 運行範例程序
python examples/demo_analysis.py
```

**優點**:
- 快速執行
- 易於自動化
- 適合批量處理

## 📊 分析內容詳解

### 技術面分析
- **價格趨勢**: 短期、中期、長期趨勢判斷
- **技術指標**: MA、MACD、RSI、布林帶等
- **支撐阻力**: 關鍵價位和交易區間
- **成交量**: 量價關係分析

### 基本面分析
- **財務狀況**: 營收、利潤、現金流分析
- **業務結構**: 各業務板塊表現
- **市場地位**: 競爭優勢和市場份額
- **增長前景**: 未來發展機會

### 市場情緒分析
- **投資者情緒**: 市場參與者態度
- **分析師評級**: 專業機構觀點
- **機構持倉**: 大資金動向
- **熱點關注**: 市場焦點話題

### 風險評估
- **宏觀風險**: 經濟環境影響
- **行業風險**: 競爭和周期性風險
- **公司風險**: 特定經營風險
- **監管風險**: 政策變化影響

### 投資建議
- **評級建議**: 買入/持有/賣出
- **目標價位**: 短期和長期目標
- **時間框架**: 投資週期建議
- **風險控制**: 止損和倉位管理

## 🛠️ 自訂分析

### 修改分析參數

您可以通過修改範例程序來自訂分析：

```python
# 在範例程序中修改
STOCK_SYMBOL = "TSLA"  # 改為您想分析的股票
ANALYSIS_DATE = "2024-06-26"  # 修改分析日期
```

### 支援的股票代碼

- **美股**: AAPL, TSLA, MSFT, GOOGL, AMZN, NVDA 等
- **指數**: SPY, QQQ, DIA 等ETF
- **其他**: 大部分在美國交易所上市的股票

## 🎯 使用技巧

### 1. 選擇合適的模型

```python
# 在配置中選擇不同的模型
"deep_think_llm": "gpt-4",         # 最高質量，適合深度分析
"quick_think_llm": "gemini-flash",  # 平衡性能，日常使用
```

### 2. 分析不同時間段

```python
# 修改分析日期來分析歷史表現
ANALYSIS_DATE = "2024-01-01"  # 年初分析
ANALYSIS_DATE = "2024-06-01"  # 半年度分析
```

### 3. 關注特定方面

您可以在提示詞中強調特定分析方向：
- 技術面分析
- 基本面分析
- 風險評估
- 行業比較

## ⚠️ 重要提醒

### 投資風險提示

1. **僅供參考**: 分析結果僅供參考，不構成投資建議
2. **風險自擔**: 投資有風險，決策需謹慎
3. **多方驗證**: 建議結合其他資訊源進行驗證
4. **專業諮詢**: 重大投資決策建議諮詢專業財務顧問

### 數據準確性

1. **實時性**: 數據可能存在延遲，請以實際市場數據為準
2. **完整性**: AI分析可能遺漏某些重要資訊
3. **準確性**: 預測和建議存在不確定性

## 🔧 故障排除

### 常見問題

1. **API密鑰錯誤**: 檢查.env檔案中的密鑰配置
2. **網路連接**: 確保網路連接正常
3. **模型響應慢**: 可以嘗試使用更快的模型

### 獲取幫助

```bash
# 查看幫助資訊
python -m cli.main help

python -m cli.main examples
```

## 📈 高級用法

### 批量分析

您可以修改程序來分析多只股票：

```python
stocks = ["AAPL", "MSFT", "GOOGL", "TSLA"]
for stock in stocks:
    # 運行分析邏輯
    analyze_stock(stock)
```

### 定期分析

設置定時任務來定期生成分析報告：

```bash
# 使用cron或Windows任務計畫程式
# 每日運行分析
0 9 * * * python examples/demo_analysis.py
```
