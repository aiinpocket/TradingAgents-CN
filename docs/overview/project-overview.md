# TradingAgents 專案概述

## 專案簡介

TradingAgents-CN 是一個基於多智慧體大語言模型（LLM）的金融交易框架，由 Tauric Research 開發並開源。本專案為中文增強版（v0.1.7），提供完整的美股支援、國際LLM整合、Docker容器化部署和專業報告匯出功能。

該專案模擬真實世界交易公司的運作模式，透過部署多個專業化的AI智慧體來協作評估市場條件並做出交易決策。

## 專案背景

### 研究動機
傳統的演算法交易系統通常依賴單一的分析模型或策略，難以應對複雜多變的金融市場。而真實的交易公司通常採用團隊協作的方式，由不同專業背景的分析師、研究員、交易員和風險管理人員共同參與決策過程。

TradingAgents 專案的核心理念是將這種人類專家團隊的協作模式數位化，透過多個專業化的AI智慧體來重現這種協作決策過程。

### 技術創新
- **多智慧體協作**: 首次將多智慧體系統應用於金融交易決策
- **專業化分工**: 每個智慧體專注於特定的分析領域
- **結構化辯論**: 透過智慧體間的辯論機制提高決策品質
- **動態風險管理**: 即時評估和調整投資風險

## 核心特性

### 1. 多維度市場分析
- **基本面分析**: 深入分析公司財務數據和基本面指標
- **技術分析**: 運用技術指標識別價格趨勢和交易訊號
- **新聞分析**: 即時監控和分析市場新聞及宏觀事件
- **情緒分析**: 分析社交媒體和投資者情緒

### 2. 智慧體協作機制
- **並行分析**: 多個分析師同時工作，提高效率
- **結構化辯論**: 看漲和看跌研究員進行觀點交鋒
- **共識形成**: 透過協商機制達成投資共識
- **風險評估**: 多層次風險管理和控制

### 3. 靈活的架構設計
- **模組化元件**: 易於擴展和客製化
- **多LLM支援**: Google AI、OpenAI、Anthropic等
- **統一配置**: 簡化的.env配置系統，啟用開關完全生效
- **智慧降級**: 資料庫不可用時自動使用檔案快取

### 4. 豐富的資料整合
- **美股數據**: FinnHub、Yahoo Finance即時數據 ✅
- **新聞數據**: Google News、財經新聞整合 ✅
- **社交數據**: Reddit情緒分析 ✅
- **資料庫支援**: MongoDB + Redis + 智慧快取 ✅

### 5. 現代化Web介面 ✅ **v0.1.2新增**
- **Streamlit介面**: 直觀的Web管理平台
- **即時進度**: 分析過程視覺化追蹤
- **配置管理**: API金鑰和系統配置管理
- **Token統計**: 即時成本追蹤和優化建議
- **響應式設計**: 支援桌面和行動端存取

TradingAgents 框架僅用於研究和教育目的。交易表現可能因多種因素而異，包括所選擇的骨幹語言模型、模型溫度、交易週期、資料品質和其他非確定性因素。

**本框架不構成財務、投資或交易建議。** 使用者在使用本框架進行任何投資決策時，應當謹慎評估風險，並諮詢專業的財務顧問。
