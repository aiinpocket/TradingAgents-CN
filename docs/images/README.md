## 📋 截圖規範

### 🖼️ 技術要求
- **格式**: PNG格式（推薦）
- **解析度**: 至少1920x1080
- **品質**: 高清，文字清晰可讀
- **大小**: 單個檔案不超過2MB

### 🎨 內容要求
- **界面完整**: 顯示完整的功能區域
- **資料真實**: 使用真實的美股股票代碼和分析結果
- **狀態清晰**: 確保界面狀態清晰可見
- **無敏感資訊**: 不包含真實的API金鑰

### 📱 建議的截圖場景
1. **主界面**: 輸入「AAPL」或「TSLA」，選擇標準分析
2. **進度界面**: 分析進行到50%左右的狀態
3. **結果界面**: 完整的分析報告，包含圖表
4. **配置界面**: 顯示Google AI、OpenAI等多個LLM提供商和模型選項

## 🚀 如何獲取截圖

### 方法1: 啟動Web應用
```bash
# 啟動Web界面
python start_web.py

# 訪問 http://localhost:8501
# 進行股票分析並截圖
```

### 方法2: Docker環境
```bash
# 啟動Docker服務
docker-compose up -d

# 訪問 http://localhost:8501
# 進行分析並截圖
```
